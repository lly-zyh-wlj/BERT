## 一、实验概述
本次实验基于 `20newsgroups` 数据集，选取 `alt.atheism`（无神论讨论）和 `soc.religion.christian`（基督教讨论）两个类别，分别使用 **BERT** 和 **GRU** 两种模型完成文本二分类任务，并对比分析两种模型的效果差异。 
## 二、实验设置 ###
1. 数据处理 - 数据集：`20newsgroups` 中的两个类别 - 数据清洗：移除邮件头、尾部和引用信息，仅保留正文 - 数据划分：训练集70%、验证集15%、测试集15%（分层抽样，保证类别分布一致） ### 2. 模型配置 - **BERT 模型**： - 预训练模型：`bert-base-uncased` - 训练轮次：2 epoch - 批次大小：8 - 学习率：2e-5 - **GRU 模型**： - 文本表示：TF-IDF 向量化（5000维） - 隐藏层维度：128 - 训练轮次：10 epoch - 学习率：0.001

## 四、结果分析
1. **准确率差异** BERT 模型的测试集准确率为 **87.41%**，相比 GRU 模型的 **81.45%**，提升了约 **6个百分点**，在本次文本分类任务中表现显著更优。
2. **性能差异原因** - **上下文理解能力**：BERT 采用双向注意力机制，能同时捕捉文本中每个词的前后上下文信息，对多义词、语境依赖的理解远强于单向的 GRU 模型。 - **预训练优势**：BERT 基于海量文本进行预训练，自带强大的通用语言知识，能更好地理解文本的深层语义；而 GRU 仅能通过训练数据学习有限的序列模式。 - **特征提取方式**：GRU 依赖 TF-IDF 词频特征，难以捕捉复杂的语义关联；BERT 则通过注意力机制自动学习文本中关键信息的权重，更适配文本分类任务。
3. **效率对比** - GRU 模型更轻量，训练速度快，对硬件要求低； - BERT 模型参数量大，训练和推理成本更高，但效果提升显著，更适合对精度要求高的场景。
## 五、结论
在本次 `20newsgroups` 二分类任务中，BERT 模型在准确率和语义理解能力上均显著优于 GRU 模型。这验证了预训练语言模型在文本分类任务中的优势，也说明对于复杂的文本语义理解任务，基于Transformer 架构的预训练模型是更优的选择。
